So What Actually Happens?

  1. The brower goes to the DNS server and finds the real address of the server that the websotes lives on.
  2. The brower sends an HTTP request message to the server asking it to send a copy of the wensite to the client.
  3. If the server approved the clients request, the server sends the client a "200 OK" message, which means "permission granted" here you go!, and ten starts sending the wensites files to the brower as a series of small chunks.
  4. The brower assembles the small chunks into a complete website and siaplay it to you.
